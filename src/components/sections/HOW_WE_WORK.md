# HowWeWorkSection - Секция "Как мы будем работать"

## Описание
Адаптивная секция с сеткой карточек, каждая из которых содержит иконку, заголовок и описание. Создана на основе дизайна из Figma.

## Структура компонентов

### HowWeWorkSection.vue
Умный компонент-секция с адаптивной сеткой

### WorkCard.vue
"Глупый" компонент карточки (иконка + заголовок + текст)

## Использование

### Базовый пример

```vue
<HowWeWorkSection
  :title="'Как мы будем работать'"
  :items="workItems"
/>
```

### Полный пример с настройками колонок

```vue
<HowWeWorkSection
  :title="'Как мы будем работать'"
  :items="workItems"
  :columns="{ mobile: 2, tablet: 3, desktop: 4 }"
/>
```

### Формат данных

```javascript
const workItems = [
  {
    icon: 'https://example.com/icon.svg', // опционально
    title: 'Старт',
    description: 'Разберу проблемы, вижн и цели. Проведу анализ и сформирую роадмап'
  },
  // ... другие элементы
]
```

## Props

### HowWeWorkSection

| Prop | Тип | По умолчанию | Описание |
|------|-----|--------------|----------|
| `title` | String | 'Как мы будем работать' | Заголовок секции |
| `items` | Array | required | Массив элементов для отображения |
| `columns` | Object | `{ mobile: 2, tablet: 3, desktop: 4 }` | Настройка количества колонок на разных breakpoints |

### WorkCard

| Prop | Тип | По умолчанию | Описание |
|------|-----|--------------|----------|
| `icon` | String | '' | URL иконки (опционально) |
| `title` | String | required | Заголовок карточки |
| `description` | String | required | Описание карточки |

## Адаптивность

- **Mobile** (< 768px): 2 колонки по умолчанию
- **Tablet** (768px - 1023px): 3 колонки по умолчанию
- **Desktop** (≥ 1024px): 4 колонки по умолчанию

Количество колонок настраивается через prop `columns`.

## Стилизация

Секция использует:
- `BaseContainer` для адаптивных отступов и фона
- `BaseHeading` (уровень 3) для заголовка
- `BaseText` (p1 и p2) для текста в карточках
- Tailwind CSS классы для сетки и отступов

## Пример из Figma

Секция точно следует дизайну из макета:
- Заголовок по центру
- Сетка карточек с равными отступами
- Иконки 32x32px
- Заголовок карточки: p1 (20px), цвет black-90
- Описание: p2 (16px), цвет black-50
