# ExpertiseSection - Секция "Команда в одном эксперте"

## Описание
Секция с карточками экспертизы, каждая из которых содержит заголовок, описание и список возможностей с чекмарками. Создана на основе дизайна из Figma.

## Структура компонентов

### ExpertiseSection.vue
Секция с заголовком, подзаголовком и адаптивной сеткой карточек

### ExpertiseCard.vue
"Глупый" компонент карточки (заголовок + описание + список с чекмарками)

## Использование

### Базовый пример

```vue
<ExpertiseSection
  :title="'Команда в одном эксперте'"
  :subtitle="'Получите не просто дизайнера...'"
  :cards="expertiseCards"
/>
```

### Формат данных

```javascript
const expertiseCards = [
  {
    title: 'Дизайнер',
    description: 'Создаю удобный и красивый продукт',
    items: [
      'Maкеты в Figma',
      'Документация в Notion',
      'Сопровождение разработки'
    ]
  },
  // ... другие карточки
]
```

## Props

### ExpertiseSection

| Prop | Тип | По умолчанию | Описание |
|------|-----|--------------|----------|
| `title` | String | 'Команда в одном эксперте' | Заголовок секции |
| `subtitle` | String | 'Получите не просто дизайнера...' | Подзаголовок секции |
| `cards` | Array | required | Массив карточек экспертизы |

### ExpertiseCard

| Prop | Тип | По умолчанию | Описание |
|------|-----|--------------|----------|
| `title` | String | required | Заголовок карточки |
| `description` | String | required | Описание карточки |
| `items` | Array | [] | Список возможностей |
| `checkmarkIcon` | String | URL из Figma | URL иконки чекмарка |

## Адаптивность сетки

- **Mobile** (< 768px): **1 колонка**
- **Tablet** (768px - 1023px): **2 колонки**
- **Desktop** (≥ 1024px): **4 колонки**

## Стилизация

### Карточки
- Белый фон (`bg-white-100`)
- Закругленные углы (`rounded-2xl`)
- Отступы: 24px (mobile), 32px (desktop)
- Высота: `h-full` для выравнивания

### Типографика
- **Заголовок карточки**: H5 (28px, medium) на desktop, P1 (20px) на mobile
- **Описание**: P2 (16px, regular), цвет black-50
- **Пункты списка**: P2 (16px), цвет black-50

### Чекмарки
- Размер: 16x16px
- Цвет: black-50
- Отступ справа: 16px

## Особенности

- Адаптивная типографика (H5 → P1 на мобильных)
- Список с иконками из Figma
- Карточки растягиваются на всю высоту
- Использует `useBreakpoints` для адаптивности
